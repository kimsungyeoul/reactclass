Node.js는 **Chrome의 V8 JavaScript 엔진** 위에서 동작하는 **Server side의 JavaScript 런타임 환경(runtime environment)** 입니다. 원래 자바스크립트는 웹 브라우저에서만 실행되었지만, Node.js는 자바스크립트를 **브라우저 외부**, 즉 **서버나 로컬 컴퓨터**에서 실행할 수 있게 해줍니다.

---

## 📌 Node.js 개요

| 항목       | 설명                                    |
| -------- | ------------------------------------- |
| 🛠 플랫폼   | 크로스 플랫폼 (Windows, macOS, Linux 모두 지원) |
| 🔧 엔진    | Chrome V8 엔진 사용 (자바스크립트를 빠르게 실행)      |
| 🔄 특징    | 비동기 I/O, 이벤트 기반 처리                    |
| 📁 주 사용처 | 웹 서버, API 서버, 실시간 채팅, 파일 서버, CLI 도구   |

---

## 💡 왜 Node.js를 사용할까?

1. **비동기 이벤트 기반 구조**

   * 하나의 스레드로 여러 작업을 동시에 처리 (non-blocking I/O)
   * 예: DB 쿼리 중에도 다른 요청을 처리 가능

2. **자바스크립트를 서버에서도 사용**

   * 프론트엔드와 백엔드 모두 같은 언어(JS) 사용 가능 → 생산성 증가

3. **npm(Node Package Manager) 생태계**

   * 수많은 오픈소스 라이브러리 사용 가능
   * `express`, `mongoose`, `react`, `next` 등

4. **빠른 실행 속도**

   * Google V8 엔진으로 자바스크립트를 기계어로 빠르게 변환하여 실행

## 🔧 Node.js 주요 구성 요소

| 구성 요소       | 설명                           |
| ----------- | ---------------------------- |
| **V8**      | 자바스크립트 엔진 (Chrome에서 가져옴)     |
| **libuv**   | 비동기 I/O 처리 라이브러리 (멀티 플랫폼 지원) |
| **C++ 바인딩** | OS와의 저수준 통신을 위해 일부 C++ 코드 포함 |
| **npm**     | 패키지 관리자 (라이브러리 설치, 배포 가능)    |

---

## 📦 대표적인 Node.js 라이브러리

| 라이브러리                 | 설명                  |
| --------------------- | ------------------- |
| `express`             | 가장 널리 쓰이는 웹 프레임워크   |
| `socket.io`           | 실시간 양방향 통신 지원 (웹소켓) |
| `mongoose`            | MongoDB 연동을 위한 ODM  |
| `axios`, `node-fetch` | HTTP 클라이언트          |
| `dotenv`              | 환경변수 관리             |

## 🖥 예제 코드: 간단한 웹 서버

```javascript
// server.js
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, { "Content-Type": "text/plain" });
  res.end("Hello, Node.js!");
});

server.listen(3000, () => {
  console.log("Server running at http://localhost:3000/");
});
```

## 📚 정리

* Node.js는 **자바스크립트를 브라우저 밖에서도 실행**할 수 있는 환경
* **비동기 이벤트 기반** 구조로 고성능 서버 개발에 적합
* **npm**으로 강력한 생태계 제공
* 실시간 서버, REST API, CLI 툴 등 **다양한 분야**에 활용 가능
